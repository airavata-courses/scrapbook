# Getting Started

###Technologies used:
Java 15

SpringBoot 2.4

RESTful API

Google drive APIs

### Project Installation
$ git clone https://github.com/airavata-courses/scrapbook.git

$ cd scrapbook-googledrive-service

$ mvn clean install

$ cd target

$ java -jar -Dspring.profiles.active=<profile name like local, dev, prod> <jar name with extension>
